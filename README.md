## ✨ 功能特点

### 📚 知识清单（KCs）

- 每章按知识点（KC）逐条整理，配有**核心概念**和**公式**
- 标签分类：`必考` / `了解` / `枢纽`，方便重点复习
- 可折叠展开，按章节分段浏览

### 📝 章节测试

- 每章内置**选择题、填空题、计算题、简答题**
- 点击选项即选，**提交**按钮即时判断对错
- 点**答案**按钮弹出参考答案（含完整公式渲染）
- 输入框可用于填空题、计算题输入答案后提交判断

### 🧮 数学公式渲染

- 基于 **KaTeX** 快速渲染 LaTeX 公式（无需安装字体或插件）
- 支持 `\hbar`、`\nabla`、`\psi`、`\lambda`、`\Delta` 等物理符号
- 支持 `\ce{}` 化学式（通过 mhchem 扩展）
- 支持 `\boldsymbol{}` 矢量符号

### 🎨 交互与视觉

- 顶部标签切换章节，内部标签切换「知识清单 / 章节测试」
- 每章独立配色，清晰的视觉分区
- 响应式设计，手机和电脑均可浏览

## 🚀 使用方式

### 方法一：直接打开（推荐）

1. 下载 `index.html` 到本地
2. 用浏览器（Chrome / Edge / Firefox 等）**双击打开**
3. 点击顶部章节标签切换内容
4. 每个章节内切换「知识清单」和「章节测试」

> ⚠ **首次打开需联网**：公式渲染依赖 KaTeX CDN（cdn.jsdelivr.net），需要有网络连接才能正常显示公式。加载过一次后浏览器会自动缓存。

### 方法二：在线访问（GitHub Pages）

## 🔧 技术栈

- **纯前端**：单 HTML 文件，无需服务器、无需数据库
- **公式渲染**：[KaTeX](https://katex.org/) v0.16.9（CDN 加载）
- **扩展插件**：auto-render（自动渲染）、mhchem（化学式支持）
- **交互脚本**：原生 JavaScript，无第三方框架依赖
