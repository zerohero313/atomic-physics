▶ §4.1 碱金属原子的光谱
KC4.1
必考
碱金属原子结构模型
概念

碱金属原子由两部分构成：一个完整的"原子实"（原子核 + 除价电子外的其余电子）和一个在原子实外运动的价电子。碱金属原子与氢原子的结构相似，因为两者核外都只有一个电子（价电子），因此氢原子理论可推广到碱金属原子。然而，碱金属的原子实是一个松散的体系，不像氢原子核那样是点电荷，会发生原子实极化以及价电子贯穿，使其光谱有别于氢原子。碱金属元素包括Li、Na、K、Rb、Cs等，它们的光谱具有相仿的结构。

KC4.2
必考
碱金属原子光谱公式
方法

碱金属原子的光谱项一般表达式为 \(T = R / (n - Δ)^2 = R / n^{*2}\)，其中 \(R\) 为里德伯常数，\(Δ\) 为量子亏损，\(n^*\) 为有效量子数。这一公式与氢原子光谱项公式 \(T = R / n^2\) 形式相似，但用有效量子数 \(n^* = n - Δ\) 替代了主量子数 \(n\)。谱线的波数由两个光谱项之差给出：\(ν = T_1 - T_2\)。不同线系的谱线波数可通过系限波数和对应谱项值计算得出。Li原子各态的光谱项值和有效量子数可通过实验数据测定，例如3s态的 \(T = 16280.5\) m\(^{-1}\)，对应的 \(n^* = 2.589\)，\(Δ = 0.4\)。

KC4.3
必考
有效量子数n*与量子亏损
概念

有效量子数 \(n^*\) 和主量子数 \(n\)、量子亏损 \(Δ\) 的关系为 \(n^* = n - Δ\)，或 \(Δ = n - n^*\)。有效量子数可由光谱项值反推：\(n^* = \sqrt{R/T}\)。量子亏损 \(Δ\) 的大小取决于角量子数 \(l\)：\(l\) 越小，轨道偏心率越大，价电子穿透原子实越深，能量降低越多，量子亏损越大。Li 原子各态的 \(Δ\) 约值：s 态约 0.40，p 态约 0.05，d 态约 0.001，f 态约 0（接近氢原子）。由 Li 光谱项数据可见，\(n^*\) 随 n 增大而逐渐接近整数，说明高激发态时碱金属原子行为趋近于氢原子。

KC4.4
必考 枢纽
四个线系(主线系、锐线系、漫线系、基线系)
事实

碱金属原子有四个谱线系：（1）主线系（Principal Series）：\(np \to 2s\) 跃迁，谱线最显著，双线结构，间隔随波数增大而减小；（2）第二辅线系/锐线系（Sharp Series）：\(ns \to 2p\) 跃迁，双线结构，间隔不变；（3）第一辅线系/漫线系（Diffuse Series）：\(nd \to 2p\) 跃迁，三线结构，第一三间隔不变，一二间隔随波数增大而减小；（4）基线系/Bergmann系（Fundamental Series）：\(nf \to 3d\) 跃迁，三线结构。不同线系对应不同的终态，每个线系的系限波数恰好是另一个线系的最大谱项。与氢光谱相比，对应 H 光谱的一个线系，碱金属有独立的几个线系。

KC4.5
必考
原子实的极化
概念

原子实由原子核和除价电子外的其余电子组成，呈球对称结构。价电子在原子实外运动时，原子实中带正电的核和核外电子云的中心会发生微小移动，形成一个电偶极子，这一现象称为原子实的极化。极化后的原子实与价电子相互作用，引起能量的变化。对同一 \(n\) 值，\(l\) 值较大（近圆轨道）的轨道引起的能量变化小，\(l\) 值越小能量变化越大。原子实的极化是碱金属能级发生 \(l\) 分裂的原因之一。

KC4.6
必考
轨道贯穿
概念

Li 原子的 s、p 能级比 H 原子相应能量低很多，说明除了原子实的极化外还存在其他原因。分析表明，s、p 轨道的偏心率很大，在接近原子实时可能会穿透（贯穿）原子实。当价电子处于原子实外运动时，有效电荷 \(Z^* \sim 1\)；当电子穿过原子实时，\(Z^* > 1\)，感受到更大的有效核电荷，从而引起更大的能量降低。轨道贯穿是 s 态能级显著低于氢原子对应能级的主要原因（对 Li 原子 s 态能级降低的贡献大于原子实极化），也是量子亏损随 \(l\) 减小而增大的重要物理机制。

KC4.7
必考 枢纽
碱金属能级分裂的原因
概念

碱金属原子能级发生 \(l\) 分裂（同一 \(n\) 值下不同 \(l\) 的能级不再简并）的原因主要有两个：（1）原子实的极化——价电子使原子实产生电偶极矩，极化后的原子实与价电子相互作用引起能量变化，\(l\) 越小效应越强；（2）轨道的贯穿——\(l\) 较小的轨道（s、p）偏心率大，电子可贯穿原子实，受到更大的有效核电荷作用，能量降低更显著。两者都使能级能量降低，\(l\) 越小降低越显著。因此同一 \(n\) 值下不同 \(l\) 的能级不再简并，表现为光谱项中量子亏损 \(Δ\) 随 \(l\) 增大而迅速减小。在考虑自旋-轨道耦合后，能级还会进一步发生 \(j\) 分裂（精细结构分裂）。

▶ §4.2 碱金属原子光谱的精细结构
KC4.8
必考
精细结构的实验特征
事实

用高分辨率仪器观察碱金属光谱时，每条谱线一般会分裂为两条或三条，而非单一的谱线，这称为光谱线的精细结构。所有的碱金属原子光谱有相仿的精细结构，一般主要可观察到四个线系。主线系和第二辅线系（锐线系）为双线结构，第一辅线系（漫线系）和基线系为三线结构。精细结构的根源在于原子的 p、d 等能层为双层结构。

KC4.9
必考
精细结构的双线/三线规律
原理

精细结构的基本规律：（1）主线系（\(np \to 2s\)）为双线结构，间隔随波数增大而减小——当 \(n\) 增大时，p 的两层能级间隔逐渐减小；（2）第二辅线系/锐线系（\(ns \to 2p\)）为双线结构，两间隔相等且保持不变——s 为单层、p 为双层能级可解释此结构；（3）第一辅线系/漫线系（\(nd \to 2p\)）为三线结构，第一三间隔（由 2p 双层间隔决定）不变，一二间隔（由 d 双层间隔决定）随波数增大而减小；（4）基线系（\(nf \to 3d\)）为三线结构，规律与漫线系相似。s 为单层结构，p、d、f 为双层结构；对同一 \(l\)，随 \(n\) 增大双层能级间隔减小；对同一 \(n\)，随 \(l\) 增大双层能级间隔减小。

▶ §4.3 碱金属原子双层结构理论
KC4.10
必考 枢纽
电子自旋假设
概念

1925 年 G. 乌伦贝克 和 S.A. 古德斯密特 提出电子不是点电荷，除了轨道角动量外还有自旋运动，具有固定的自旋角动量 \(P_s = \sqrt{s(s+1)}ħ\)，其中自旋量子数 \(s = 1/2\)。自旋角动量在 \(z\) 方向的分量为 \(P_{sz} = m_s ħ\)，\(m_s = \pm 1/2\)。电子的自旋磁矩为 \(\boldsymbol{μ}_s = -g_s \frac{e}{2m} \boldsymbol{P}_s\)，其中 \(g_s = 2\) 为电子自旋的 Land\'e 因子（电子轨道 Land\'e 因子 \(g_l = 1\)）。自旋假设成功解释了碱金属光谱的精细结构、斯特恩-盖拉赫实验等关键实验现象。

KC4.11
必考
自旋-轨道耦合
原理

价电子绕原子实运动相当于原子实绕价电子运动，原子实的运动使价电子感受到一个磁场的存在。电子自旋磁矩在此磁场中受到力矩作用，产生自旋-轨道耦合，使原子能量发生变化。总角动量 \(\boldsymbol{P}_j = \boldsymbol{P}_l + \boldsymbol{P}_s\)，总角量子数 \(j = l \pm 1/2\)（对 \(l > 0\)）。自旋-轨道耦合能量 \(Δ E_{ls} ∝ \frac{Z^{*4}}{n^3 l(l+1/2)(l+1)}\)。由此可知：（1）对同一 \(l\)，\(n\) 越大间隔越小（\(∝ 1/n^3\)）；（2）对同一 \(n\)，\(l\) 越大间隔越小；（3）\(Z\)（原子序数）越大间隔越大（\(∝ Z^4\)），这解释了 Na 的精细结构分裂比 Li 更大的原因。

KC4.12
必考
碱金属能级双层结构
概念

自旋-轨道耦合能取决于总角量子数 \(j\)。\(l = 0\)（s 态）时，\(j\) 只能取 \(1/2\)，只有一个可能值，不发生能级分裂，因此 s 能级为单层。\(l > 0\)（p、d、f 等）时，\(j\) 可取 \(l + 1/2\) 和 \(l - 1/2\) 两个值，对应不同能量，能级分裂为双层。双层能级差为 \(Δ E = \frac{1}{2} \zeta (2l+1)\)，其中 \(\zeta\) 为自旋-轨道耦合常数。这就是碱金属原子能级双层结构的由来：p 能级分裂为 \(n^2P_{1/2}\) 和 \(n^2P_{3/2}\)，d 能级分裂为 \(n^2D_{3/2}\) 和 \(n^2D_{5/2}\)，等等。

KC4.13
必考
选择定则(Δl=±1, Δj=0,±1)
原理

▶ §5.1 氦原子光谱与能级
KC5.1
必考
氦原子光谱特征(两套谱系)
事实

氦原子光谱具有两套独立的谱线系：单线结构（仲氦）和多线结构（正氦）。每套谱线系又可分为主线系、第一辅线系、第二辅线系和基线系等。单线结构为单层结构，多线结构为三层结构。这是氦原子区别于氢原子的最显著特征，源于两个价电子自旋态的不同组合。

KC5.2
必考
正氦与仲氦
概念

氦原子因两电子自旋取向不同而分为两种：正氦（自旋平行，三重态）和仲氦（自旋反平行，单态）。正氦具有三层结构（多线结构），仲氦为单层结构（单线结构）。两种氦在光谱上表现出完全不同的特征，但它们是同一元素的不同自旋态。

KC5.3
必考
氦原子能级结构
概念

氦原子能级分为两套：单态（仲氦）和三重态（正氦）。同一主量子数 \(n\) 下，三重态能级低于单态能级。氦能级结构比氢复杂得多，与电子自旋状态密切相关。氦的基态为 \(1s^2\) 构成的 \(^1S_0\)，是第一激发能最大的元素之一。

▶ §5.2 电子耦合与原子态
KC5.4
必考 枢纽
电子组态
概念

由两个或多个价电子所处状态的组合称为电子组态。对于双价电子原子，一般一个电子处于基态，另一个处于基态或激发态，记为 \(nln'l'\)。与基态最靠近的态称为第一激发态。若两个电子的主量子数 \(n\) 不同，能级间隔大；若角量子数 \(l\) 不同，能级间隔则较小。

KC5.5
必考
电子间六种相互作用
事实

两电子之间存在六种电磁相互作用：两电子轨道间的相互作用（\(l_1\)–\(l_2\)耦合）、两电子自旋间的相互作用（\(s_1\)–\(s_2\)耦合）、每个电子自身的自旋-轨道耦合（\(l_1\)–\(s_1\)和\(l_2\)–\(s_2\)）、以及一个电子的轨道与另一个电子自旋间的交叉相互作用（\(l_1\)–\(s_2\)和\(l_2\)–\(s_1\)，作用较弱）。不同相互作用强弱对比决定了耦合类型。

KC5.6
必考 枢纽
L-S耦合(罗素-桑德斯耦合)
方法

适用于轻元素。在电子间的静电相互作用（\(l_1\)–\(l_2\)和\(s_1\)–\(s_2\)）强于自旋-轨道耦合时采用。先将各电子的轨道角动量耦合成总轨道角动量 \(L = \sum l_i\)，自旋角动量耦合成总自旋 \(S = \sum s_i\)，再通过自旋-轨道相互作用将 \(L\) 和 \(S\) 耦合成总角动量 \(J = L+S, L+S-1, \dots, |L-S|\)。

KC5.7
必考
原子态表示法
概念

原子态用光谱项符号表示为 \(^{2S+1}L_J\)。其中左上角 \(2S+1\) 为自旋多重度，\(L\) 为总轨道角动量（\(L=0,1,2,3,\dots\) 分别对应 \(S,P,D,F,\dots\)），右下角 \(J\) 为总角动量。例如 \(^3P_2\) 表示 \(S=1\)（三重态）、\(L=1\)、\(J=2\) 的原子态。

KC5.8
必考
洪特定则(洪特定则)
原理

用于判断同一电子组态各原子态能级高低：①自旋多重度 \(2S+1\) 最大的能级最低；②多重度相同时，总轨道角动量 \(L\) 最大的能级最低；③\(S\) 和 \(L\) 均相同时，\(J\) 最小的能级最低（正常次序，适用于同科电子数 \(\le\) 闭壳层一半）或 \(J\) 最大的能级最低（反常次序）。仅适用于 \(L\)-\(S\) 耦合类型。

KC5.9
了解
朗德间隔定则
原理

在一个多重能级（\(S>0\)）结构中，相邻两个能级之间的间隔与相关的两个 \(J\) 值中较大的那个成正比：\(Δ E(J, J-1) ∝ J\)。该定则为判断能级结构是否符合 \(L\)-\(S\) 耦合提供了定量依据。\(j\)-\(j\) 耦合的能级间隔不满足此定则。

KC5.10
必考
j-j耦合
方法

适用于重元素（高 \(Z\)），自旋-轨道相互作用（\(l_i\)–\(s_i\)）强于电子间相互作用。先对每个电子分别合成其总角动量 \(j_i = l_i + s_i\)（取 \(j_i = |l_i \pm s_i|\)），再将各电子的 \(j_i\) 耦合成原子的总角动量 \(J = \sum j_i\)。\(j\)-\(j\) 耦合与 \(L\)-\(S\) 耦合形成的原子态数目相同，但能级结构和选择定则不同。

KC5.11
必考
辐射跃迁选择定则
原理

偶极辐射跃迁必须满足宇称改变规则：跃迁只能发生在不同宇称的状态之间。\(L\)-\(S\) 耦合下：\(Δ l = \pm 1\)（单电子），\(Δ L = 0, \pm 1\)，\(Δ S = 0\)，\(Δ J = 0, \pm 1\)（但 \(J=0 \to 0\) 禁戒）。\(j\)-\(j\) 耦合下：\(Δ j = 0, \pm 1\)。宇称由各电子 \(l\) 值之和的奇偶性判断。

KC5.17
必考
原子基态的确定
方法

确定原子基态三步法：①由原子序数 \(Z\) 确定基态电子组态（按 \(1s \to 2s \to 2p \to 3s \to 3p \to 4s \to 3d \to \dots\) 填充）；②写出电子组态形成的所有原子态（满壳层贡献 \(^1S_0\)）；③用洪特定则和朗德间隔定则找出能量最低的原子态。满壳层和满次壳层的轨道、自旋及总角动量均为零。

KC5.18
了解
多种耦合方式的比较
概念

\(L\)-\(S\) 耦合和 \(j\)-\(j\) 耦合是两种极端情形：①\(L\)-\(S\) 耦合适用于轻元素（电子间库仑作用为主），能级间隔符合朗德定则；②\(j\)-\(j\) 耦合适用于重元素（自旋-轨道作用为主），不符合朗德定则；③两者形成的原子态数目相同；④中间区域（中重元素）实际耦合介于二者之间，非典型类型。

▶ §5.3 泡利不相容原理
KC5.12
必考 枢纽
泡利不相容原理
原理

1925年由泡利提出：在费米子组成的系统中，不能有两个或两个以上的粒子处于完全相同的状态。描述电子态需四个量子数 \((n, l, m_l, m_s)\)，即不能有两个电子的这四个量子数全部相同。该原理解释了 \(1s1s\) 组态只能形成 \(^1S_0\) 而不能形成 \(^3S_1\)，也是元素周期律的物理基础。

KC5.13
了解
同科电子
概念

具有相同主量子数 \(n\) 和角量子数 \(l\) 的电子称为同科电子。受泡利原理限制，同科电子的 \(m_l\) 或 \(m_s\) 不能全同，因此同科电子形成的原子态比非同科电子少。确定同科电子原子态的方法有Slater表格法和Slater图解法。例如 \(np^2\) 组态只能形成 \(^1S\)、\(^1D\)、\(^3P\) 三个态。

KC5.16
必考
简并度与状态数
事实

第 \(n\) 主壳层可容纳的最多电子数为 \(\sum_{l=0}^{n-1} 2(2l+1) = 2n^2\)。每个次壳层 \(l\) 有 \(2l+1\) 个轨道磁量子数 \(m_l\) 取值，每个 \(m_l\) 又有自旋向上和向下两个态（\(m_s = \pm 1/2\)），故每个次壳层可容纳 \(2(2l+1)\) 个电子。例如 \(n=3\) 壳层最多容纳 \(2 × 3^2 = 18\) 个电子。

▶ §5.4 元素周期表
KC5.14
必考
元素周期律的物理本质
概念

元素性质的周期性源于核外电子组态的周期性排列。泡利原理限制了每个壳层可容纳的电子数（第 \(n\) 主壳层最多 \(2n^2\) 个），电子按能量最小原理依次填充各壳层。每完成一个主壳层的填充即完成一个周期，满壳层对应惰性气体元素，不满壳层决定元素的化学性质。

KC5.15
必考 枢纽
壳层结构与电子排布
方法

壳层分为主壳层（\(n=1,2,3,\dots\) 对应 \(K,L,M,\dots\)）和次壳层（\(l=0,1,2,3,\dots\) 对应 \(s,p,d,f,\dots\)）。每个次壳层可容纳 \(2(2l+1)\) 个电子。电子填充次序为 \(1s \to 2s \to 2p \to 3s \to 3p \to 4s \to 3d \to 4p \to 5s \to 4d \to 5p \to 6s \to 4f \to 5d \to 6p \to 7s \to 5f \to \dots\)，其中存在能级交错现象。

▶ §6.1 外磁场对原子的作用
KC6.1
必考 枢纽
原子磁矩(轨道+自旋)
概念

原子磁矩由三部分组成：电子轨道磁矩、电子自旋磁矩和原子核磁矩。由于核质量远大于电子质量（约1836倍），核磁矩比电子磁矩小约三个数量级，通常可忽略。轨道磁矩方向与轨道角动量方向相反（电子带负电）：\( \vec{μ}_l = -\frac{e}{2m} \vec{P}_l \)。自旋磁矩方向与自旋角动量方向相反：\( \vec{μ}_s = -\frac{e}{m} \vec{P}_s \)。

KC6.2
必考
朗德g因子
方法

朗德g因子是连接角动量与磁矩的关键量。对于单电子原子：\( g = 1 + \frac{j(j+1) + s(s+1) - l(l+1)}{2j(j+1)} \)。其中\( l \)为轨道角动量量子数，\( s \)为自旋量子数（\( s=1/2 \)），\( j \)为总角动量量子数。若\( s=0 \)则\( g=1 \)；若\( l=0 \)则\( g=2 \)。多电子原子需先通过LS耦合求出总轨道角动量\( L \)和总自旋角动量\( S \)，再由\( L \)和\( S \)耦合出总角动量\( J \)，代入类似公式计算。

KC6.3
必考
单电子原子的总磁矩
方法

单电子原子中，电子的轨道磁矩\( \vec{μ}_l \)和自旋磁矩\( \vec{μ}_s \)都在绕总角动量\( \vec{P}_j \)旋进，它们的合成矢量\( \vec{μ} \)的大小和方向随时间变化，不是定值。但\( \vec{μ} \)在\( \vec{P}_j \)方向上的投影\( μ_j \)是定值，对外界产生实际效果的就是这个投影。总磁矩的表达式为：\( μ_j = g · \sqrt{j(j+1)} · μ_B \)，其中\( μ_B = \frac{eħ}{2m} \)为玻尔磁子。

KC6.4
必考
多电子原子的磁矩
方法

多电子原子磁矩的计算步骤：(1)通过LS耦合求出电子的总轨道角动量\( L \)和总自旋角动量\( S \)；(2)由量子数\( L \)和\( S \)耦合出总角动量量子数\( J \)；(3)代入朗德g因子公式计算；(4)原子总磁矩\( μ_J = g · \sqrt{J(J+1)} · μ_B \)。其中\( L \)、\( S \)、\( J \)通过洪特规则和泡利不相容原理确定。对于多电子原子，可将最后一个电子与前\( n-1 \)个电子分开考虑，分别计算后再求和。

KC6.5
必考
原子受外磁场作用的附加能量
原理

原子处于外磁场中时，磁矩受磁场力矩作用产生附加能量。弱磁场下：\( Δ E = M · g · μ_B · B \)，其中\( M \)为磁量子数，取值为\( -J, -J+1, ..., J-1, J \)共\( 2J+1 \)个。一个\( J \)能级分裂成\( 2J+1 \)层，相邻能层间隔为\( g · μ_B · B \)。强磁场下（帕邢-Back效应）：\( Δ E = μ_B B (m_L + 2m_S) \)，\( m_L \)和\( m_S \)分别独立取值。弱场中\( L \)和\( S \)先耦合成\( J \)再与外场作用；强场中\( L \)和\( S \)分别直接与外场作用。

▶ §6.2 物质的磁性
KC6.12
必考
物质的磁性分类
事实

物质的磁性分为抗磁性、顺磁性、铁磁性、反铁磁性和亚铁磁性等。磁性可由磁化率\( \chi \)区分：\( \chi<0 \)为抗磁性物质，\( \chi>0 \)为顺磁性物质，铁磁性物质\( \chi\gg1 \)。单原子物质的磁性取决于原子的磁矩；由分子构成的物质则取决于分子磁矩。任何物质都具有磁性，在不均匀磁场中都会受到磁力作用。

KC6.13
必考
抗磁性
概念

抗磁性起源于磁场对电子轨道运动的作用。原子中所有电子绕外磁场旋进形成电环流，因电子带负电，电环流产生的感生磁矩与磁场方向相反，对外显现抗磁性。抗磁性普遍存在于所有原子或分子中，但只有在总磁矩为零（\( J=0 \)）的情况下才会显现出来。电子轨道运动受温度影响小，因此抗磁性受温度影响小，磁化率\( \chi<0 \)。

KC6.14
必考
顺磁性
概念

顺磁性是具有磁矩的原子（\( J≠0 \)）在外磁场中各种取向的平均效果。热运动使原子按玻尔兹曼分布，低能级（\( μ_z>0 \)）分布的原子更多，因此平均磁矩方向与磁场一致，对外显现顺磁性。顺磁磁化率满足居里定律：\( \chi = C/T \)，其中\( C \)为居里常数。温度升高时高低能级原子数差别减小，顺磁性减弱。顺磁性物质也具有抗磁性，只是较强的顺磁性掩盖了抗磁性。

KC6.15
了解
铁磁性
概念

铁磁性的起因是物质中存在固有磁矩，且固有磁矩之间存在强烈的交换耦合。与顺磁性不同，铁磁物质中存在许多总磁矩不为零的"磁畴"——各磁畴的磁矩方向各不相同，无外场时对外不显示宏观磁化。加外磁场时，磁畴的磁矩方向强烈取向，使物质强烈磁化。铁磁性与固体结构相关，磁化率\( \chi\gg1 \)。

▶ §6.3 顺磁共振
KC6.16
必考
顺磁共振(电子自旋共振)
概念

顺磁共振：顺磁原子处于磁场中能级发生等间距分裂，在垂直于恒定磁场方向加上特定频率的交变磁场时，相邻能级间发生共振跃迁吸收能量的现象。共振条件：\( hν = gμ_B B \)（相邻能级间隔）。轨道磁矩因频繁变化平均贡献仅约1%，总磁矩绝大部分来自电子自旋，因此顺磁共振也称为电子自旋共振（ESR）。固体或液体样品中因邻近原子影响，能级间隔不再相等，常出现多个共振峰；核磁矩的影响还可能产生波谱的精细结构。

▶ §6.4 塞曼效应
KC6.6
必考 枢纽
塞曼效应(正常/反常)
概念

塞曼效应是指光源放在磁场中时，所发光谱线分裂成多条且每条谱线都是偏振光的现象，由P. 塞曼于1896年发现。正常塞曼效应发生在自旋\( s=0 \)的情况下，谱线分裂成3条，间隔为一个洛伦兹单位（\( L = \frac{eB}{4\pi m c} \)）。反常塞曼效应发生在自旋\( s≠0 \)的情况下，由于\( g \)因子不为1，谱线分裂成多于3条。塞曼效应证实了原子磁矩的空间量子化，可用于测量电子的荷质比和天体磁场。

KC6.7
必考
正常塞曼效应
方法

正常塞曼效应条件为\( s=0 \)、\( g_1=g_2=1 \)，谱线分裂成3条，频率间隔为一个洛伦兹单位。跃迁选择定则：\( Δ M = 0 \)产生\( \pi \)线（电矢量平行于磁场\( B \)），\( Δ M = \pm 1 \)产生\( \sigma \)线（电矢量垂直于\( B \)）。沿垂直于\( B \)方向观察可见3条平面偏振光；沿\( B \)方向观察，\( \pi \)线不出现，\( \sigma \)线为圆偏振光（\( \sigma^+ \)右旋、\( \sigma^- \)左旋）。经典算例：Cd原子6438Å谱线（\( ^1D_2→{}^1P_1 \)）分裂为3条。

KC6.8
必考
反常塞曼效应
方法

反常塞曼效应发生在自旋\( s≠0 \)的原子中，由于\( g \)因子不为1，谱线分裂成多于3条。磁能级分裂间隔为\( Δ E = M g μ_B B \)，跃迁频率差为\( Δν = (M_2g_2 - M_1g_1)L \)。经典算例：Na D1线（\( ^2P_{1/2}→{}^2S_{1/2} \)，5896Å）分裂为4条谱线；Na D2线（\( ^2P_{3/2}→{}^2S_{1/2} \)，5890Å）分裂为6条谱线。这是由于上下能级的\( g \)因子不同，裂距不相等所致。

KC6.11
了解
塞曼效应的偏振特性
事实

塞曼效应中，沿光的传播方向反方向（逆着光传播方向）观察：若电矢量作顺时针转动称为右偏振光；作逆时针转动称为左偏振光。\( \pi \)线（\( Δ M=0 \)）的电矢量平行于\( B \)，沿\( B \)方向观察不出现；\( \sigma^+ \)线（\( Δ M=+1 \)）为右旋圆偏振光；\( \sigma^- \)线（\( Δ M=-1 \)）为左旋圆偏振光。光子的角动量方向和电矢量旋转方向成右手螺旋法则：左偏振光的传播方向和角动量方向相同，右偏振光的传播方向与角动量方向相反。

▶ §6.5 帕邢-Back效应
KC6.9
必考
帕邢-巴克效应
概念

帕邢-巴克效应是强磁场中的塞曼效应。物理条件：外磁场足够强，使得自旋-轨道耦合被破坏，\( L \)和\( S \)不再耦合形成\( J \)，而是分别独立绕外磁场进动。此时反常塞曼效应被帕邢-巴克效应取代，谱线分裂恢复到3条（类似于正常塞曼效应），因此说它"趋于正常塞曼效应"。从弱场到强场是一个连续过渡过程，中间区域能级结构复杂。

KC6.10
了解
强磁场下的能级分裂
方法

强磁场下原子附加能量为\( Δ E = μ_B B (m_L + 2m_S) \)，\( m_L \)和\( m_S \)分别独立取值。弱磁场下为\( Δ E = M g μ_B B \)，总角动量量子数\( J \)起主导作用。二者本质区别：弱场中\( L \)和\( S \)先耦合成\( J \)再与外场作用；强场中\( L \)和\( S \)直接分别与外场作用，\( L-S \)耦合被破坏。高分辨率下可观察到每一条\( \sigma \)成分还包含靠得很近的2条谱线，这是因为强场中仍存在较弱的\( L-S \)耦合剩余效应。

▶ §7.1 X射线的发现及其波性质
KC7.1
必考
X射线的发现与基本性质
事实

1895年11月8日，德国物理学家伦琴在研究阴极射线时意外发现了一种穿透力极强的新射线，命名为X射线(后也称伦琴射线)。X射线波长约为 \(1\ \text{Å}\) 左右，人眼不可见，但能使荧光物质发光、使气体电离、使照相底片感光，并能透过可见光不能透过的物体。根据穿透能力，X射线分为硬X射线(波长短、穿透力强)和软X射线(波长较长、穿透力较弱)。X射线的发现是19世纪末20世纪初物理学三大发现之一，标志着现代物理学的产生，并直接推动了放射性现象的发现。

KC7.2
必考
X射线的产生
概念

X射线一般通过高速电子打击金属靶产生。产生装置为X射线管，是一个高真空玻璃管(管内压强约 \(10^{-6}\ \text{mmHg}\) 至 \(10^{-8}\ \text{mmHg}\))，内有两个电极：阴极(K)发射电子，阳极(A)为金属靶。阴极与阳极之间加有数万至十几万伏的高压电场，电子被加速后高速撞击阳极靶，电子急剧减速时将动能转化为辐射能，从而发射X射线。阳极靶材一般选用高熔点金属，如钨、钼、铜等，以承受电子撞击产生的高温。调节管电压可改变X射线的波长分布。

KC7.3
了解
X射线的偏振证明
概念

1906年，巴克拉发现X射线的偏振现象，证明了X射线是横波(电磁波)。实验方法为二次散射偏振实验：非偏振X射线沿z方向入射到第一散射体(如石蜡)，若X射线是横波，散射体内的电子将在垂直于传播方向的x-y平面内做受迫振动，从而发出偏振波。从x方向观察时，可得到y方向偏振的X射线。再让该偏振X射线通过第二散射体：在z方向能观察到强的散射X射线，而在y方向则观察不到，从而验证了X射线的横波性质。

KC7.4
必考 枢纽
布拉格公式与X射线衍射
原理

1912年，劳厄发现X射线通过晶体时产生衍射现象，证明X射线是一种波长较短的电磁波。随后布拉格父子成功解释了X射线晶体衍射机理，提出了布拉格公式：\(2d\sinθ = nλ\)。其中 \(d\) 为晶面间距，\(θ\) 为掠射角(入射线与晶面的夹角)，\(n\) 为衍射级数(正整数)，\(λ\) 为X射线波长。利用该公式，通过测量衍射角 \(θ\) 可计算X射线波长或晶体的晶面间距。布拉格父子于1913年设计出第一台X射线分光计，并利用它发现了特征X射线。该原理在晶体结构分析和X射线光谱学中有重要应用。

▶ §7.2 X射线产生的机制
KC7.5
必考
X射线连续谱与韧致辐射
概念

X射线连续谱是高速电子撞击金属靶时，电子在靶原子核库仑场作用下急剧减速而产生的电磁辐射，称为韧致辐射(bremsstrahlung)。由于电子与靶原子核的碰撞是随机的，不同电子损失的能量不同，辐射出的光子频率连续变化，形成从某最短波长到长波方向的连续谱分布。连续谱的形状与管电压有关，电压越高则谱线向短波方向扩展，强度也越大。连续谱叠加了标识谱线，共同构成X射线谱。

KC7.6
必考
最短波长(截止波长)
原理

连续谱中存在一个最短波长(截止波长) \(λ_{\min}\)，对应于电子动能全部转化为一个光子的能量。由能量守恒：\(eV = hν_{\max} = \frac{hc}{λ_{\min}}\)，因此 \(λ_{\min} = \frac{hc}{eV}\)。式中 \(e\) 为电子电荷，\(V\) 为管电压，\(h\) 为普朗克常量，\(c\) 为光速。当管电压 \(V = 50\ \text{kV}\) 时，\(λ_{\min} ≈ 0.0248\ \text{nm}\)。该公式是量子论早期的重要验证之一，说明X射线光子的能量来源于电子动能的转化。

KC7.7
必考 枢纽
X射线标识谱
概念

X射线标识谱(特征X射线)是叠加在连续谱上的线状谱，其谱线波长由靶材元素种类决定，与入射电子能量无关(仅需能量足够大以激发该谱系)，因此称为标识谱。每种元素都有自己独特的标识X射线谱，反映了原子内层电子的能级结构。标识谱的发现是布拉格父子于1913年利用X射线分光计完成的，它揭示了原子内层结构的奥秘，为理解原子结构提供了重要手段。

KC7.8
必考
K系、L系等线系
事实

X射线标识谱由多个线系组成，波长最短的是K系，此外还有L系、M系等。各元素的标识谱具有相似结构：K系是最内层(n=1)以外的各层电子跃迁到K层(n=1)的结果；L系是n=2以外的各层电子跃迁到L层(n=2)的结果，以此类推。K系谱线波长最短(能量最高)，L系次之。不同元素的同一条线系结构相似，但波长随原子序数增大而变短。标识谱不具有周期性，因其来源于内层电子跃迁，而内层电子都是填满的。

KC7.9
必考
内层电子跃迁机制
概念

X射线标识谱来源于原子内层电子的跃迁。要产生标识谱，必须先在原子内层产生电子空位，主要有两种方法：(1)高速电子与靶原子发生非弹性碰撞，将内层电子击出；(2)原子吸收能量足够高的光子，通过光电效应使内层电子电离。内层产生空位后，外层电子跃迁填充该空位，同时释放出能量等于两能级差的光子，即标识X射线。跃迁遵循单电子辐射跃迁选择定则：\(Δ l = \pm 1\)，\(Δ j = 0, \pm 1\)。K层电离所需能量最大，L层次之。

KC7.10
必考
莫塞莱定律
原理

莫塞莱于1913年系统研究了各种元素的特征X射线，发现特征X射线频率的平方根与原子序数Z成线性关系，即莫塞莱定律：\(\sqrt{ ν} = a(Z - b)\)。对于K系谱线，有 \(\sqrt{ ν_K} = \sqrt{\frac{3}{4}Rc}(Z - 1)\)，其中 \(R\) 为里德伯常量，\(c\) 为光速，\(b\) 为屏蔽常数(对K系 \(b ≈ 1\)，L系 \(b ≈ 7.4\))。莫塞莱定律的重要意义在于：(1)确立了原子序数Z作为元素周期表排列顺序的物理依据；(2)为当时尚未发现的元素提供了预测手段；(3)验证了玻尔原子理论中能级公式的正确性。

▶ §7.3 X射线的吸收
KC7.11
必考
X射线的吸收(朗伯-比尔定律)
原理

X射线通过物质时，强度因吸收和散射而衰减，服从朗伯-比尔定律：\(I = I_0 e^{-μ x}\)，其中 \(I_0\) 为入射强度，\(I\) 为穿过厚度 \(x\) 后的强度，\(μ\) 为衰减系数。衰减由两个过程引起：吸收(光电效应等)和散射(仅改变方向使原方向强度减弱)。引入原子衰减常数 \(μ_a\)、原子吸收系数 \(\tau_a\) 和原子散射系数 \(\sigma_a\)，有 \(μ = \tau + \sigma\)。吸收系数与波长的关系近似为 \(μ/ ho ∝ λ^3 Z^4\)，即波长越短(硬X射线)贯穿能力越强，原子序数越高的物质吸收越强。

KC7.12
必考
吸收限
概念

X射线吸收谱中，吸收系数随波长减小而平缓降低，但当波长减小到某些特定值时，吸收系数会突然增大，这些突变位置称为吸收限。吸收限的产生原因是：当X射线光子的能量恰好等于原子某内层电子的电离能时，发生强烈的共振吸收(光电效应)，使吸收系数骤增。每种元素有多个吸收限，分别对应K层、L层、M层等不同内层电子的电离阈值。K吸收限的波长最短(能量最高)，L吸收限次之，以此类推。吸收限的位置是元素的特征，可通过测量吸收限来确定元素种类。

▶ §8.1 原子核物理的研究对象
KC8.1
必考
原子核物理发展史
事实

1896年贝克勒尔发现铀的放射现象，标志着原子核物理学的开端。1898年居里夫妇发现钋和镭，1899年发现α、β射线，1900年发现γ射线。1911年卢瑟福提出原子核式模型，1919年首次实现人工核反应(用α粒子从氮核打出质子)。1932年查德威克发现中子，海森伯随即提出原子核由质子和中子组成。1934年约里奥-居里夫妇发现人工放射性。1939年哈恩和斯特拉斯曼发现重核裂变，同年玻尔和惠勒提出裂变的液滴模型理论。1942年费米建成第一座核反应堆，1954年苏联建成第一座原子能发电站。我国1958年建成第一座重水反应堆，1964年成功试爆原子弹，1967年试爆氢弹成功。

KC8.2
必考
原子核的组成(核子)
概念

原子核由质子(proton)和中子(neutron)组成，两者统称为核子(nucleon)。质子带正电(电荷量为 \(+e\))，中子呈电中性。核子质量：质子质量 \(m_p = 1.007276 \text{u}\)，中子质量 \(m_n = 1.008665 \text{u}\)。原子核表示为 \(\ce{^A_Z X}\)，其中 \(Z\) 为质子数(原子序数)，\(N\) 为中子数，\(A = Z + N\) 为质量数(核子数)。在发现中子之前，人们曾将原子核误认为是质子和电子的组成体，但这无法解释核自旋等问题。

KC8.3
必考
核素、同位素、同量异位素
概念

核素(nuclide)指具有特定质子数Z和中子数N的原子核类型。同位素(isotope)是质子数相同(Z相同)但质量数A不同(中子数不同)的核素，它们在周期表中位置相同，化学性质基本相同。同量异位素(isobar)是质量数A相同但质子数Z不同的核素，即核子总数相同但质子-中子组成不同。例如 \(\ce{^14_6C}\) 和 \(\ce{^14_7N}\) 是同量异位素，而 \(\ce{^1_1H}\)、\(\ce{^2_1H}\)、\(\ce{^3_1H}\) 互为同位素。

KC8.4
必考
核素图与稳定区
事实

核素图是以中子数N为横坐标、质子数Z为纵坐标的二维图，每个格点代表一种核素。图中稳定核素几乎全部落在一条光滑曲线附近，称为核素稳定区(核素半岛)。轻核区稳定核素满足 \(N ≈ Z\)；随Z增大，稳定线逐渐向 \(N > Z\) 方向偏离，因为质子间的库仑斥力随Z增大而增强，需要更多中子提供核力以维持稳定。稳定线上侧为缺中子核区，下侧为丰中子核区。天然存在约300个核素(其中280多个稳定)，人工制造的放射性核素达1600多个，总计约2000个核素。

▶ §8.2 原子核的基本性质
KC8.5
必考 枢纽
原子核质量与结合能
原理

原子核由质子和中子组成，其质量小于组成核子的质量之和，这个差值称为质量亏损。结合能 \(E_B\) 是将原子核分解为单个核子所需的最小能量，由质能关系 \(E_B = Δ m · c^2\) 决定。若原子核质量为 \(M\)，质子质量 \(m_p\)，中子质量 \(m_n\)，则结合能为 \(E_B = [Z m_p + (A-Z) m_n - M]c^2\)。原子质量单位 \(1 \text{u} = 1.66054 × 10^{-27} \text{kg}\)，对应的能量为 \(931.5 \text{MeV}\)。

KC8.6
必考
质量亏损与质能关系
原理

质量亏损(mass defect)是指原子核的实际质量小于其组成核子(质子和中子)质量之和的现象。质量亏损的物理本质由爱因斯坦质能关系 \(E = mc^2\) 解释：核子结合成原子核时释放出一部分能量(结合能)，根据相对论，这部分能量的释放对应着质量的减少。质量亏损 \(Δ m\) 与结合能 \(E_B\) 的关系为 \(E_B = Δ m · c^2\)。例如氦-4核(\(\ce{^4_2He}\))的质量亏损 \(Δ m = 0.0304 \text{u}\)，对应结合能约 \(28.3 \text{MeV}\)。

KC8.7
必考 枢纽
比结合能(平均结合能)曲线
概念

比结合能(平均结合能)是每个核子的平均结合能，即 \(E_B/A\)。比结合能曲线以质量数A为横坐标、比结合能为纵坐标，呈现以下规律：(1)中等质量核(\(A ≈ 40-120\))的比结合能最大，约 \(8.5 \text{MeV/核子}\)，最稳定；(2)轻核和重核的比结合能较小，重核约 \(7.5 \text{MeV/核子}\)；(3)质量数 \(A > 30\) 的核素比结合能变化不大，说明核力具有饱和性；(4)曲线有周期性起伏，最大值出现在 \(A\) 为4的倍数处(如 \(\ce{^4_2He}\)、\(\ce{^{12}_6C}\)、\(\ce{^{16}_8O}\))。比结合能曲线是解释核裂变和核聚变释放能量的重要依据。

KC8.9
必考
原子核自旋
概念

原子核具有角动量，总角动量又称为原子核自旋(nuclear spin)，是核内所有质子和中子的轨道角动量与自旋角动量的矢量和。核自旋量子数 \(I\) 的取值与质量数 \(A\) 的奇偶性有关：\(A\) 为偶数时 \(I\) 取整数(包括0)，\(A\) 为奇数时 \(I\) 取半整数。例如 \(\ce{^1_1H}\) 的 \(I = 1/2\)，\(\ce{^4_2He}\) 的 \(I = 0\)，\(\ce{^{14}_7N}\) 的 \(I = 1\)。核自旋在核磁共振(NMR)技术中有重要应用。

KC8.10
了解
原子核磁矩
概念

原子核中质子带正电，其运动产生磁场，使原子核具有磁矩。中子虽不带电，但其内部电荷分布和自旋也对核磁矩有贡献。核磁矩表示为 \(\boldsymbol{μ}_I = g_I β_N \boldsymbol{I}\)，其中 \(β_N = eħ/(2m_p) = 5.0508 × 10^{-27} \text{J/T}\) 为核磁子，\(g_I\) 为朗德因子(需实验测定)，\(\boldsymbol{I}\) 为核自旋。核磁矩比原子磁矩小约三个数量级(核磁子比玻尔磁子小约1836倍)，因此在原子物理中通常可忽略。常见核磁矩值：质子 \(+2.79255 β_N\)，中子 \(-1.91280 β_N\)，氘核 \(+0.857348 β_N\)。

▶ §8.4 原子核的结构模型
KC8.8
必考
半经验质量公式(液滴模型)
原理

基于液滴模型，原子核结合能的半经验质量公式(贝特-魏茨泽克公式)包含以下项：(1)体积能 \(a_V A\)，正比于核子数A，反映核力的饱和性；(2)表面能 \(-a_S A^{2/3}\)，表面核子受核力不饱和，减小结合能；(3)库仑能 \(-a_C Z(Z-1)/A^{1/3}\)，质子间库仑斥力降低稳定性；(4)对称能 \(-a_A (A-2Z)^2/A\)，质子和中子数不对称时降低结合能；(5)成对能 \(\delta\)，与核子奇偶性有关(偶偶核最稳定，奇奇核最不稳定)。公式可准确计算大量核素的质量，预言核稳定性和衰变方式。

KC8.13
必考
原子核结构模型(液滴/壳层)
概念

原子核结构模型是对原子核进行近似、唯象描述的理论框架。液滴模型将原子核类比为带电荷的液滴，核子间核力类比于液体分子间的短程吸引力，质子库仑斥力类比于液滴电荷排斥，能很好地解释核结合能、核裂变等整体性质。壳层模型则将核子视为在平均势场中独立运动，考虑自旋-轨道耦合后能正确预言幻数(2、8、20、28、50、82、126)，解释原子核的稳定性周期变化。此外还有费米气体模型和集体运动模型，分别侧重不同方面的核性质。

KC8.14
必考
液滴模型
概念

液滴模型(liquid-drop model)由玻尔和惠勒于1939年提出。该模型将原子核类比于一个带电荷的不可压缩液滴：核子间核力类比于液体分子间的短程吸引力，使核聚集成球形；质子间的库仑斥力类比于液滴电荷之间的排斥作用，使核趋向于分裂。液滴模型成功导出了半经验质量公式，解释了原子核结合能的规律，并应用于核裂变机制的分析——复核的集体振荡导致形变，表面张力试图恢复球形，库仑力促使分裂，两者竞争决定裂变是否发生。该模型主要描述原子核的集体行为。

KC8.15
必考
壳层模型与幻数
概念

壳层模型(shell model)由梅耶和延森于1949年提出(获1963年诺贝尔奖)。该模型假设每个核子在由其他核子形成的平均势场(如无限深球方势阱或谐振子势)中独立运动，并考虑核子的自旋-轨道耦合。求解薛定谔方程得到核子的能级次序，当质子数或中子数为幻数(magic number)2、8、20、28、50、82、126时，核子正好填满完整壳层，原子核特别稳定。幻数核的特点包括：丰度高、结合能大、中子俘获截面小、第一激发态能量高等。壳层模型成功解释了核自旋、核磁矩和同位素丰度等实验事实。

▶ §8.5 原子核的放射衰变
KC8.16
必考 枢纽
放射性衰变(α/β/γ)
概念

放射性衰变是不稳定核素自发转变为另一种核素的过程。主要衰变方式：(1)α衰变：放出氦核 \(\ce{^4_2He}\)，子核质子数减2、质量数减4，如 \(\ce{^{238}_{92}U ightarrow ^{234}_{90}Th + α}\)。α粒子能量分立，反映核能级的量子化。(2)β衰变：包括β⁻衰变(中子转质子，放出电子和反中微子，Z增1)、β⁺衰变(质子转中子，放出正电子和中微子，Z减1)和电子俘获(核俘获核外电子使质子转中子)。β能谱连续，泡利为此提出中微子假说。(3)γ衰变：原子核从激发态跃迁到低能态时发射γ光子(电磁辐射)，不改变Z和A。γ跃迁遵守宇称和角动量守恒。还存在内转换(IC，能量直接传给核外电子)和同质异能跃迁等特殊形式。

KC8.17
必考
衰变规律与半衰期
原理

放射性衰变服从指数衰变律：\(N = N_0 e^{-λ t}\)，其中 \(N_0\) 为初始核数，\(λ\) 为衰变常数(单位时间衰变概率)，\(t\) 为时间。半衰期 \(T_{1/2}\) 是核数衰减到一半所需时间：\(T_{1/2} = \ln 2 / λ = 0.693 / λ\)。平均寿命 \(\tau = 1/λ = T_{1/2} / \ln 2\)。半衰期是每种放射性核素的标识特征。对于级联衰变(放射系)，母核衰变成子核，子核仍有放射性，逐级衰变直至稳定核素。自然界存在四个放射系：铀系(\(A=4n+2\))、锕系(\(A=4n+3\))、钍系(\(A=4n\))、镎系(\(A=4n+1\))。

▶ §8.6 原子核反应
KC8.18
必考
原子核反应
概念

原子核反应是用具有一定能量的粒子轰击原子核使其性质发生变化的过程。核反应方程一般表示为 \(A(a,b)B\)，即靶核(入射粒子，出射粒子)剩余核，例如卢瑟福1919年实现的第一个人工核反应 \(\ce{^{14}_7N}(α, p)\ce{^{17}_8O}\)。反应能 \(Q\) 定义为反应前后总质量差对应的能量：\(Q = (M_A + m_a - M_B - m_b)c^2\)，\(Q > 0\) 为放能反应，\(Q < 0\) 为吸能反应。核反应截面 \(\sigma\) 描述反应概率，定义为每个靶核的有效作用面积。核反应机制包括库仑散射、核势散射、形成复核、削裂反应、掇拾反应等。按入射粒子能量分为低能(\(<100\text{MeV}\))、中能和高能核反应。

▶ §8.7 原子核裂变与聚变
KC8.19
必考
核裂变
概念

核裂变(nuclear fission)是重原子核分裂为两个或多个中等质量核的过程。1939年哈恩和斯特拉斯曼发现中子轰击铀核产生裂变。典型裂变反应：\(\ce{^{235}_{92}U + n ightarrow ^{236}_{92}U^* ightarrow X + Y + (2\sim3)n}\)。裂变释放大量能量(每个 \(\ce{^{235}U}\) 裂变约 \(200\text{MeV}\))，来源于比结合能曲线——重核比结合能(约 \(7.6\text{MeV/核子}\))低于中等质量核(约 \(8.5\text{MeV/核子}\))。每次裂变伴生约2.5个中子，可维持链式反应。\(\ce{^{235}U}\) 可由热中子(\(0.03eV\))诱发裂变，而 \(\ce{^{238}U}\) 需快中子(\(>1\text{MeV}\))。临界体积是维持自持链式反应的最小体积，利用中子减速剂(石墨、重水)可提高反应效率。

KC8.20
必考
核聚变
概念

核聚变(nuclear fusion)是两个轻原子核结合成较重原子核的过程。由于比结合能曲线，轻核(如氘、氚)聚合成中等质量核时比结合能增大，释放能量。典型聚变反应：\(\ce{^2_1H + ^3_1H ightarrow ^4_2He + n + 17.6MeV}\)。聚变需极高温度(约 \(10^8 \sim 10^{10}\text{K}\))使核子获得足够动能克服库仑势垒，故又称热核反应。劳森判据给出自持聚变条件：等离子体温度、密度和约束时间需满足一定关系。太阳等恒星的能源来自聚变(质子-质子循环和碳循环)。人造聚变包括氢弹(惯性约束)和可控聚变(磁约束，如托卡马克装置)。中国EAST装置(东方超环)在2017年实现了 \(101.2s\) 稳态高约束等离子体运行。

KC8.21
了解
核能应用
事实

核能的应用主要包括以下几方面：(1)核能发电——1954年苏联建成世界上第一座原子能发电站，利用核裂变链式反应放热发电。我国1958年建成第一座重水型原子反应堆。(2)核武器——原子弹(基于核裂变，如1945年投于广岛的 \(\ce{^{235}U}\) 弹"小男孩")、氢弹(裂变触发聚变，三相弹加 \(\ce{^{238}U}\) 外壳增强威力)、中子弹(增强聚变辐射，减弱冲击波和放射性沾染)。(3)放射性同位素应用——医学(放射治疗、示踪诊断)、工业(无损检测、辐照改性)、农业(辐射育种)。(4)聚变能前景——可控热核聚变是人类的终极能源目标，国际热核聚变实验堆(ITER)和中国聚变工程实验堆(CFETR)正在推进。
